New Features & Enhancements:

  • Implemented new and more secured encryption algorithm to work with new version of Plus Utility (V2.4.3 or higher). 

WARNING:

  1. After GS105Ev2 is upgraded to FW version V1.5.0.4 or higher, it cannot work with Plus Utility version prior to V2.4.3.
     Please download the latest Plus Utility from     http://www.netgear.com/support/product/PCU
  2. Once GS105Ev2 FW has been upgraded to V1.5.0.4, you need to login to web GUI to upgrade boot loader. GS105Ev2 has 2 different chip versions and they have different boot loader version. Firmware will detect the IC version automatically and prompt you for the required boot loader version on the upgrade GUI page.

Upgrade Procedure:

  1. Download the new firmware 1.5.0.4 from NETGEAR product support Website and store it in local PC, and remember its location in the PC’s file system. Unzip it to get the switch firmware image “GS105Ev2_V1.5.0.4.bin”.

    Download Link: http://www.downloads.netgear.com/files/GDC/GS105EV2/GS105Ev2_V1.5.0.4.zip

  2. There are two methods to upgrade the new firmware from PC to the switch.
    • HTTP
    • ProSAFE Plus Utility

    Below are the steps to upgrade the firmware via HTTP, which is the most convenient method. For the other method, please check the IG and SW User Manual for details.

  3. To use HTTP method, go to GUI page “Maintenance > Firmware Upgrade".
  4. Click Enter Loader Mode button and it will go to Firmware Upgrade page.
  5. Click the Browse button and locate and select the new firmware image file “GS105Ev2_V1.5.0.4.bin”.
  6. Click the Apply button.
  7. When the upgrade is complete, your switch restarts. The upgrade process typically takes about three minutes.
  8. Once FW upgrading completes, please login to switch by web GUI to upgrade boot loader.
  9. When login to the switch, it will go to Bootloader Upgrade page.
  10. It will show which boot loader version you should use and click Browse button and locate and select the correct boot loader version and click APPLY.
  11. Once boot loader upgrading is complete, the switch will go back to login page.  

WARNING:

To avoid the risk of corrupting the firmware, do not interrupt the upgrade. For example, do not turn off the switch or disconnect it.
